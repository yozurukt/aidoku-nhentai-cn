# Aidoku nhentai Source

这是 [Aidoku Community Sources](https://github.com/Aidoku-Community/sources) 的 fork，仅包含 **nhentai** 源。并会同步上游源的更新。

主要针对中文用户习惯进行了以下增强：

- **标签双向翻译**：可以显示汉化后的标签，还支持在设置（收藏/屏蔽）里直接输入中文标签。 
- **收藏标签**：可以在设置里添加常用的标签，方便快速筛选。（支持中文/英文标签）
- **排序优化**：增加了标签列表排序选项，支持按“热门程度”排序。

翻译数据来源于 [EhTagTranslation](https://github.com/EhTagTranslation/Database)。

## 安装方法

 打开 Aidoku 应用,进入「设置」标签页，在「来源列表」页面中添加地址：`https://yozurukt.github.io/aidoku-nhentai-cn/index.json`
